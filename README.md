# App Shelf

A personal media tracker for iOS. Organize the games, shows, movies, books, and music you're playing, watching, reading, or listening to — all in one place.

## Features

- **Custom shelves** — comes with defaults like *Currently Playing*, *Backlog*, and *Finished*, or create your own
- **Rich item details** — cover art, star ratings, mood tags (vibes), notes, and start/finish dates
- **Year in Review** — stats dashboard with completions, average shelf time, top vibes, and media type breakdowns
- **Home screen widgets** — small, medium, and lock screen sizes showing your current shelf at a glance
- **Automatic dates** — moving an item to *Finished* sets the finish date automatically

## Stack

- Swift / SwiftUI
- SwiftData
- WidgetKit + AppIntents
- iOS 17+
